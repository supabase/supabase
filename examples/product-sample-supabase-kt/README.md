### Setup
Android Studio with SDK 30 or above
Open `local.properties` file, add these
```kotlin
API_KEY=YOUR_SUPABASE_API_KEY
SECRET=YOUR_SUPABASE_SECRET
SUPABASE_URL=YOUR_SUPABASE_URL

```
