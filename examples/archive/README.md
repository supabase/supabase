# Supabase Examples Archive

## supabase-js v1

You can find the supabase-js v1 examples at [github.com/supabase/examples-archive](https://github.com/supabase/examples-archive).

## Overview

This directory contains archived examples from earlier versions of Supabase. These examples demonstrate how to build applications using Supabase with different client libraries and frameworks.

### Current Examples

- **supabase-js v1**: Legacy examples using the first version of the Supabase JavaScript client library. These examples are preserved for reference and historical purposes.

### Migration Guide

If you're using supabase-js v1, we recommend upgrading to the latest version for improved performance, security updates, and new features. Check out the [official documentation](https://supabase.com/docs) for the current best practices.
