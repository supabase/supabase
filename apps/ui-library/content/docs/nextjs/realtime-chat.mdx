## Introduction

The Realtime Chat component is designed to allow users to send and receive messages in real-time within a shared room. It can be used to build real-time collaborative applications with chat functionality.

## Usage

The Realtime Chat component is designed to be used in a room. It can be used to build real-time collaborative applications with chat functionality. Add the `<RealtimeChat />` component to your page and it will render a chat interface that syncs messages across all users in the room.

```tsx
'use client'

import { RealtimeChat } from '@/registry/default/blocks/realtime-chat/components/realtime-chat'

export default function ChatPage() {
  return (
    <RealtimeChat
      roomName="general"
      username="john_doe"
      avatar="https://example.com/avatar.jpg" // optional
    />
  )
}
```

## Database Setup

This component requires a Supabase table called `messages` with the following structure:

```sql
create table messages (
  id uuid default uuid_generate_v4() primary key,
  room text not null,
  content text not null,
  user jsonb not null,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null
);

-- Enable realtime for the messages table
alter publication supabase_realtime add table messages;
```

## Props

| Prop     | Type              | Description                                                                       |
| -------- | ----------------- | --------------------------------------------------------------------------------- |
| roomName | string            | The name of the chat room. Users in the same room will see each other's messages. |
| username | string            | The display name for the current user.                                            |
| avatar   | string (optional) | URL for the user's avatar image.                                                  |
