# Storybook

A storybook app that can be used to develop our design system in isolation.

## Add stories for a component

Create a `MyComponent.stories.tsx` file, usually next to the component itself.
Example: [MetricCard.stories.tsx](../../packages/ui-patterns/src/MetricCard/MetricCard.stories.tsx)

## Adding a documentation page for a component

Create a `MyComponent.doc.mdx` file, usually next to the component itself.
Example: [MetricCard.doc.mdx](../../packages/ui-patterns/src/MetricCard/MetricCard.doc.mdx)

## Adding a general documentation page

Create a `MyPage.mdx` file in `apps/storybook/src`.
Example: [Introduction.mdx](./src/index.mdx)
