  <StepHikeCompact.Step step={2}>

    <StepHikeCompact.Details title="Install agent skills (optional)">

    </StepHikeCompact.Details>

    <StepHikeCompact.Code>


  <StepHikeCompact.Step step={3}>

    <StepHikeCompact.Details title="Set up your database">

    Run the following in your project's [SQL Editor](/dashboard/project/_/sql/new) to create an `instruments` table with sample data and a policy that allows public read access.

    </StepHikeCompact.Details>

    <StepHikeCompact.Code>

      ```sql SQL_EDITOR
      -- Create the table
      create table instruments (
        id bigint primary key generated always as identity,
        name text not null
      );

      -- Insert sample data into the table
      insert into instruments (name)
      values
        ('violin'),
        ('viola'),
        ('cello');

      grant select on public.instruments to anon;

      alter table instruments enable row level security;

      create policy "public can read instruments"
      on public.instruments
      for select to anon
      using (true);
      ```

    </StepHikeCompact.Code>

  </StepHikeCompact.Step>

  <StepHikeCompact.Step step={4}>

    <StepHikeCompact.Details title="Create a React app">

    Use a [Vite](https://vitejs.dev/guide/) template to create a React app.

    </StepHikeCompact.Details>

    <StepHikeCompact.Code>

      ```bash
      npm create vite@latest my-app -- --template react
      ```

    </StepHikeCompact.Code>

  </StepHikeCompact.Step>

  <StepHikeCompact.Step step={5}>
    <StepHikeCompact.Details title="Install the Supabase client library">

    `supabase-js` provides a convenient interface for querying Supabase from a React app. Navigate to your app and install it.

    </StepHikeCompact.Details>

    <StepHikeCompact.Code>

      ```bash
      cd my-app && npm install @supabase/supabase-js
      ```

    </StepHikeCompact.Code>

  </StepHikeCompact.Step>

  <StepHikeCompact.Step step={6}>
    <StepHikeCompact.Details title="Declare Supabase Environment Variables">

    Create a `.env.local` file and add your Supabase URL and publishable key. You can find these in the project's Connect panel.

    <Button variant="primary" asChild>
      <a href="/dashboard/project/_?showConnect=true&connectTab=frameworks&framework=react">Open Connect panel</a>
    </Button>

    </StepHikeCompact.Details>

    <StepHikeCompact.Code>

      ```text
      VITE_SUPABASE_URL=<SUBSTITUTE_SUPABASE_URL>
      VITE_SUPABASE_PUBLISHABLE_KEY=<SUBSTITUTE_SUPABASE_PUBLISHABLE_KEY>
      ```

    </StepHikeCompact.Code>

  </StepHikeCompact.Step>

  <StepHikeCompact.Step step={7}>
    <StepHikeCompact.Details title="Query Supabase data from React">

    Replace the contents of `src/App.jsx` with the following. The `createClient` call connects your app to Supabase and queries the `instruments` table.

    </StepHikeCompact.Details>
    <StepHikeCompact.Code>

      ```js name=src/App.jsx
      import { useEffect, useState } from "react";
      import { createClient } from "@supabase/supabase-js";

      const supabase = createClient(
        import.meta.env.VITE_SUPABASE_URL,
        import.meta.env.VITE_SUPABASE_PUBLISHABLE_KEY
      );

      function App() {
        const [instruments, setInstruments] = useState([]);

        useEffect(() => {
          getInstruments();
        }, []);

        async function getInstruments() {
          const { data } = await supabase.from("instruments").select();

          setInstruments(data ?? []);
        }

        return <pre>{JSON.stringify(instruments, null, 2)}</pre>;
      }

      export default App;
      ```

    </StepHikeCompact.Code>

  </StepHikeCompact.Step>

  <StepHikeCompact.Step step={8}>
    <StepHikeCompact.Details title="Start the app">

    Run the development server, go to http://localhost:5173 in a browser and you should see the list of instruments.

    </StepHikeCompact.Details>

    <StepHikeCompact.Code>

      ```bash
      npm run dev
      ```

    </StepHikeCompact.Code>

  </StepHikeCompact.Step>
</StepHikeCompact>
