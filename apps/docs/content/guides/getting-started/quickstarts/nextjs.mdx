<StepHikeCompact>

  <StepHikeCompact.Step step={1}>

    <StepHikeCompact.Details title="Create a Supabase project">

    Sign up and create a new Supabase project in the [Dashboard](https://supabase.com/dashboard).

    </StepHikeCompact.Details>

  </StepHikeCompact.Step>

  <StepHikeCompact.Step step={2}>

    <StepHikeCompact.Details title="Install agent skills (optional)">

    [Agent Skills](/docs/guides/ai-tools/ai-skills) give AI coding agents Supabase-specific context so they can help you build faster. Install them in your project with:

    </StepHikeCompact.Details>

    <StepHikeCompact.Code>

      ```bash
      npx skills add supabase/agent-skills
      ```

    </StepHikeCompact.Code>

  </StepHikeCompact.Step>

  <StepHikeCompact.Step step={3}>

    <StepHikeCompact.Details title="Set up your database">

    Run the following in your project's [SQL Editor](/dashboard/project/_/sql/new) to create an `instruments` table with sample data and a policy that allows public read access.

    </StepHikeCompact.Details>

    <StepHikeCompact.Code>

      ```sql SQL_EDITOR
      -- Create the table
      create table instruments (
        id bigint primary key generated always as identity,
        name text not null
      );

      -- Insert sample data into the table
      insert into instruments (name)
      values
        ('violin'),
        ('viola'),
        ('cello');

      grant select on public.instruments to anon;

      alter table instruments enable row level security;

      create policy "public can read instruments"
      on public.instruments
      for select to anon
      using (true);
      ```

    </StepHikeCompact.Code>

  </StepHikeCompact.Step>

  <StepHikeCompact.Step step={4}>

    <StepHikeCompact.Details title="Create a Next.js app">

    Use the `create-next-app` command with the `with-supabase` template to create a Next.js app pre-configured with `supabase-js`, [cookie-based Auth](/docs/guides/auth/server-side/creating-a-client?queryGroups=package-manager&package-manager=npm&queryGroups=framework&framework=nextjs&queryGroups=environment&environment=server), [TypeScript](https://www.typescriptlang.org/), and [Tailwind CSS](https://tailwindcss.com/).

    </StepHikeCompact.Details>

    <StepHikeCompact.Code>

      ```bash
      npx create-next-app -e with-supabase
      ```

    </StepHikeCompact.Code>

  </StepHikeCompact.Step>

  <StepHikeCompact.Step step={5}>
    <StepHikeCompact.Details title="Declare Supabase Environment Variables">

    Rename `.env.example` to `.env.local` and add your Supabase URL and publishable key. You can find these in the project's Connect panel.

    <Button variant="primary" asChild>
      <a href="/dashboard/project/_?showConnect=true&connectTab=frameworks&framework=nextjs">Open Connect panel</a>
    </Button>

    </StepHikeCompact.Details>

    <StepHikeCompact.Code>

      ```text
      NEXT_PUBLIC_SUPABASE_URL=<SUBSTITUTE_SUPABASE_URL>
      NEXT_PUBLIC_SUPABASE_PUBLISHABLE_KEY=<SUBSTITUTE_SUPABASE_PUBLISHABLE_KEY>
      ```

    </StepHikeCompact.Code>

  </StepHikeCompact.Step>

  <StepHikeCompact.Step step={6}>
    <StepHikeCompact.Details title="Query Supabase data from Next.js">

    Create a new file at `app/instruments/page.tsx` and populate it with the following. The generated `createClient` helper gives you a Supabase client to query the `instruments` table.

    </StepHikeCompact.Details>

    <StepHikeCompact.Code>

    <$CodeTabs>

      ```ts name=app/instruments/page.tsx
      import { createClient } from "@/lib/supabase/server";
      import { Suspense } from "react";

      async function InstrumentsData() {
        const supabase = await createClient();
        const { data: instruments } = await supabase.from("instruments").select();

        return <pre>{JSON.stringify(instruments, null, 2)}</pre>;
      }

      export default function Instruments() {
        return (
          <Suspense fallback={<div>Loading instruments...</div>}>
            <InstrumentsData />
          </Suspense>
        );
      }
      ```

    </$CodeTabs>

    </StepHikeCompact.Code>

  </StepHikeCompact.Step>

  <StepHikeCompact.Step step={7}>
    <StepHikeCompact.Details title="Start the app">

    Run the development server, go to http://localhost:3000/instruments in a browser and you should see the list of instruments.

    </StepHikeCompact.Details>

    <StepHikeCompact.Code>

      ```bash Terminal
      npm run dev
      ```

    </StepHikeCompact.Code>

  </StepHikeCompact.Step>

</StepHikeCompact>
