  <StepHikeCompact.Step step={5}>
    <StepHikeCompact.Details title="Query data from the app">

      Open `my_app/my_app.py` and replace its contents with the following. The `State` class holds the data, the `load_instruments` event handler fetches it asynchronously from Supabase, and `on_load` triggers the handler when the page is opened.

    </StepHikeCompact.Details>
    <StepHikeCompact.Code>

      ```python name=my_app/my_app.py
      import os
      import reflex as rx
      from supabase import acreate_client, AsyncClient
      from dotenv import load_dotenv

      load_dotenv()

      _supabase: AsyncClient | None = None


      async def get_supabase() -> AsyncClient:
          global _supabase
          if _supabase is None:
              _supabase = await acreate_client(
                  os.environ.get("SUPABASE_URL"),
                  os.environ.get("SUPABASE_PUBLISHABLE_KEY"),
              )
          return _supabase


      class State(rx.State):
          instruments: list[dict] = []

          async def load_instruments(self):
              supabase = await get_supabase()
              response = await supabase.table("instruments").select("*").execute()
              self.instruments = response.data


      def index() -> rx.Component:
          return rx.container(
              rx.heading("Instruments"),
              rx.foreach(
                  State.instruments,
                  lambda instrument: rx.text(instrument["name"]),
              ),
          )


      app = rx.App()
      app.add_page(index, on_load=State.load_instruments)
      ```

    </StepHikeCompact.Code>

  </StepHikeCompact.Step>
