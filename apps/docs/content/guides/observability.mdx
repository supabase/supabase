Supabase bundles the telemetry primitives, Studio dashboards, and export hooks you need to monitor projects, optimize performance, and resolve security issues. Use built-in reports and advisors when you want guidance in Studio, or forward metrics/logs into your preferred observability stack with a few clicks.

## Observability Products and Features

Pick the workflow that matches where you spend time today. The cards below link to the detailed guides for each surface.

      guide](/docs/guides/telemetry/metrics/grafana-self-hosted) and install Prometheus using the
