Telemetry is the raw data layer behind Supabase Observability. Every tool—Studio Reports, Query Performance, Advisors, and external dashboards—consumes one of three signals: logs, metrics, or traces.

Supabase normalizes each signal so you can stay inside Studio or export them to your preferred vendor using the same credentials and APIs.
