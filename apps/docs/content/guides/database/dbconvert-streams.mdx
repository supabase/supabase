If you do not have DBConvert Streams, you can install it from its [website](https://streams.dbconvert.com/install).

## What is DBConvert Streams?

DBConvert Streams is a GUI tool for working with PostgreSQL databases. You can use it to connect to your Supabase database via SSL.
