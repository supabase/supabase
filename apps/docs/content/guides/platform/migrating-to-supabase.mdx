Learn how to migrate to Supabase from another database service.

## Which migration do I need?

If you're looking for migration guidance, choose the guide that matches your use case:

- [**Migrating to Supabase**](/docs/guides/platform/migrating-to-supabase) (this page) — Migrate data from another database service (PostgreSQL, MySQL, Firebase, etc.) to Supabase
- [**Migrating within Supabase**](/docs/guides/platform/migrating-within-supabase) — Migrate data between two Supabase projects
- [**Database Migrations**](/docs/guides/deployment/database-migrations) — Manage versioned schema changes for your Supabase project using the CLI
