## Authenticate with the Supabase API

To interact with data in database tables, you can use the client library that wrap the auto-generated Data API endpoints. 

To do this, you need to authenticate using the Project URL and key from [the project **Connect** dialog](/dashboard/project/\_?showConnect=true&connectTab={{ .tab }}&framework={{ .framework }}).

[Read the API keys docs](/docs/guides/getting-started/api-keys) for a full explanation of all key types and their uses.
