This reference documents every module and function available in Supabase's Elixir library, [supabase_potion](https://hex.pm/packages/supabase_potion). You can use supabase_potion together with its companion packages to interact with your Postgres database, listen to database changes, invoke Deno Edge Functions, build login and user management functionality, and manage large files.

The Elixir SDK is composed of several Hex packages:

- [`supabase_potion`](https://hex.pm/packages/supabase_potion) - Core client library
- [`supabase_postgrest`](https://hex.pm/packages/supabase_postgrest) - PostgREST query builder
- [`supabase_auth`](https://hex.pm/packages/supabase_auth) - Authentication
- [`supabase_storage`](https://hex.pm/packages/supabase_storage) - File storage
- [`supabase_realtime`](https://hex.pm/packages/supabase_realtime) - Realtime subscriptions
- [`supabase_functions`](https://hex.pm/packages/supabase_functions) - Edge Functions

<Admonition type="note">

The Elixir client library is created and maintained by the Supabase community, and is not an official library. Please be tolerant of areas where the library is still being developed, and — as with all the libraries — feel free to contribute wherever you find issues.

Huge thanks to official maintainer, [Zoey de Souza Pessanha](https://github.com/zoedsoupe).

</Admonition>
