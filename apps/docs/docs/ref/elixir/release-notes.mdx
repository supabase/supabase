The Elixir client library is actively maintained by the Supabase community.

For the latest releases, changelogs, and version updates, visit the [supabase-ex GitHub releases page](https://github.com/supabase-community/supabase-ex/releases).

## Key Features

- **Modular Architecture**: Install only the service modules you need (auth, storage, postgrest, functions, realtime)
- **Lazy Query Execution**: Build complex queries with a fluent interface
- **Type Safety**: Leverage Elixir's type system and pattern matching
- **OTP Integration**: Self-managed clients run as Agent processes in your supervision tree
- **Pluggable HTTP Client**: Default Finch adapter with support for custom implementations
