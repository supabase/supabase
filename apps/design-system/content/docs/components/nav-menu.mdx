## Usage

NavBar is exclusively used as sub-navigation within PageLayout.

```tsx
import { NavMenu, NavMenuItem } from 'ui'
```

```tsx
<NavMenu>
  <NavMenuItem active={true}>
    <Link href="#">Overview</Link>
  </NavMenuItem>
  <NavMenuItem active={false}>
    <Link href="#">Documentation</Link>
  </NavMenuItem>
</NavMenu>
```
