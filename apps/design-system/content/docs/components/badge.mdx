Badge displays compact metadata such as labels, tags, and inline qualifiers. Its purpose should be self-evident based on surrounding context.

<ComponentPreview name="badge-demo" peekCode />

## Usage

```tsx
import { Badge } from '@/components/ui/badge'
```

```tsx
<Badge variant="default">Internal</Badge>
```

Use `Badge` for generic metadata where the label should stay visually quiet.

Use `ContextBadge` for holistic uppercase context labels such as plan, environment, or header pills.

Use `StatusBadge` for operational lifecycle and process status.

Use `StateBadge` for binary enabled and disabled state.
