## Sub-components

- `PageHeader.Root` - Root container with size variants (`default`, `small`, `large`, `full`)
- `PageHeader.Breadcrumb` - Breadcrumb navigation wrapper
- `PageHeader.Icon` - Icon container positioned left of title
- `PageHeader.Summary` - Container for title and description
- `PageHeader.Title` - Primary page heading (h1)
- `PageHeader.Description` - Supporting text below title
- `PageHeader.Aside` - Container for action buttons
- `PageHeader.Footer` - Container for tab navigation (NavMenu)
