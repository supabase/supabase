## Sub-components

- `PageSection.Root` - Root container with orientation variants (`horizontal` or `vertical`)
- `PageSection.Summary` - Container for section title and description
- `PageSection.Title` - Section heading (h2)
- `PageSection.Description` - Supporting text below section title
- `PageSection.Aside` - Container for section-level actions
- `PageSection.Content` - Container for the main section content

## Orientation Variants

- `vertical` - Content stacks vertically (default)
- `horizontal` - Summary and content arranged horizontally on larger screens
