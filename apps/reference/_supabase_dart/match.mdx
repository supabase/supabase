Finds all rows whose columns match the specified `query` object.

<Tabs
  defaultValue="dart"
  groupId="reference/dart"
  values={[{ label: 'Dart', value: 'dart' }]}>

<TabItem value="dart">

```dart
final res = await supabase
  .from('cities')
  .select('name, country_id')
  .match({'name': 'Beijing', 'country_id': 156})
  .execute();
```

</TabItem>

</Tabs>
