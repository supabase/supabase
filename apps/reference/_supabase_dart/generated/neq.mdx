Finds all rows whose value on the stated `column` doesn't match the specified `value`.

```dart
final data = await supabase
  .from('cities')
  .select('name, country_id')
  .neq('name', 'The shire');
```

## Examples

### With `select()`

```dart
final data = await supabase
  .from('cities')
  .select('name, country_id')
  .neq('name', 'The shire');
```

### With `update()`

```dart
final data = await supabase
  .from('cities')
  .update({ 'name': 'Mordor' })
  .neq('name', 'San Francisco');
```

### With `delete()`

```dart
final data = await supabase
  .from('cities')
  .delete()
  .neq('name', 'Mordor');
```

### With `rpc()`

```dart
// Only valid if the Stored Procedure returns a table type.
final data = await supabase
  .rpc('echo_all_cities')
  .neq('name', 'Lagos');
```
