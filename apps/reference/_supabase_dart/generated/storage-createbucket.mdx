Creates a new Storage bucket

```dart
final res = await supabase
  .storage
  .createBucket('avatars');
```

## Notes

- Policy permissions required:
  - `buckets` permissions: `insert`
  - `objects` permissions: none

## Examples

### Create bucket

```dart
final res = await supabase
  .storage
  .createBucket('avatars');
```
