Finds all rows whose value in the stated `column` matches the supplied `pattern` (case sensitive).

```dart
final res = await supabase
  .from('cities')
  .select('name, country_id')
  .like('name', '%la%')
  .execute();
```

## Examples

### With `select()`

```dart
final res = await supabase
  .from('cities')
  .select('name, country_id')
  .like('name', '%la%')
  .execute();
```

### With `update()`

```dart
final res = await supabase
  .from('cities')
  .update({ 'name': 'Mordor' })
  .like('name', '%la%')
  .execute();
```

### With `delete()`

```dart
final res = await supabase
  .from('cities')
  .delete()
  .like('name', '%la%')
  .execute();
```

### With `rpc()`

```dart
// Only valid if the Stored Procedure returns a table type.
final res = await supabase
  .rpc('echo_all_cities')
  .like('name', '%la%')
  .execute();
```
