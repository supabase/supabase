Performs a DELETE on the table.

```dart
final res = await supabase
  .from('cities')
  .delete()
  .match({ 'id': 666 })
  .execute();
```

## Notes

- `delete()` should always be combined with [Filters](/docs/reference/dart/using-filters) to target the item(s) you wish to delete.

## Examples

### Delete records

```dart
final res = await supabase
  .from('cities')
  .delete()
  .match({ 'id': 666 })
  .execute();
```
