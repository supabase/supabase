Removes all objects inside a single bucket.

```dart
final res = await supabase
  .storage
  .emptyBucket('avatars');
```

## Notes

- Policy permissions required:
  - `buckets` permissions: `select`
  - `objects` permissions: `select` and `delete`

## Examples

### Empty bucket

```dart
final res = await supabase
  .storage
  .emptyBucket('avatars');
```
