Limits the result to rows within the specified range, inclusive.

```dart
final res = await supabase
  .from('cities')
  .select('name, country_id')
  .range(0,3)
  .execute();
```

## Examples

### With `select()`

```dart
final res = await supabase
  .from('cities')
  .select('name, country_id')
  .range(0,3)
  .execute();
```
