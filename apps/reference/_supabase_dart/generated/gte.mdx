Finds all rows whose value on the stated `column` is greater than or equal to the specified `value`.

```dart
final data = await supabase
  .from('cities')
  .select('name, country_id')
  .gte('country_id', 250);
```

## Examples

### With `select()`

```dart
final data = await supabase
  .from('cities')
  .select('name, country_id')
  .gte('country_id', 250);
```

### With `update()`

```dart
final data = await supabase
  .from('cities')
  .update({ 'name': 'Mordor' })
  .gte('country_id', 250);
```

### With `delete()`

```dart
final data = await supabase
  .from('cities')
  .delete()
  .gte('country_id', 250);
```

### With `rpc()`

```dart
// Only valid if the Stored Procedure returns a table type.
final data = await supabase
  .rpc('echo_all_cities')
  .gte('country_id', 250);
```
