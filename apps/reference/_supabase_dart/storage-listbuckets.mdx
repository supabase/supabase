  groupId="reference/dart"

<TabItem value="dart">

```dart
final res = await supabase
  .storage
  .listBuckets()
```

</TabItem>

</Tabs>

## Notes

- Policy permissions required:
  - `buckets` permissions: `select`
  - `objects` permissions: none
