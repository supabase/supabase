Downloads a file.

<Tabs
  defaultValue="dart"
  groupId="reference/dart"
  values={[{ label: 'Dart', value: 'dart' }]}>

<TabItem value="dart">

```dart
final res = await supabase
  .storage
  .from('avatars')
  .download('avatar1.png');
```

</TabItem>

</Tabs>

## Notes

- Policy permissions required:
  - `buckets` permissions: none
  - `objects` permissions: `select`
