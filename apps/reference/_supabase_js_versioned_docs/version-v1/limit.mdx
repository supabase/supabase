Limits the result with the specified `count`.

```js
const { data, error } = await supabase
  .from('cities')
  .select('name, country_id')
  .limit(1)
```

## Examples

### With `select()`

```js
const { data, error } = await supabase
  .from('cities')
  .select('name, country_id')
  .limit(1)
```

### With embedded resources

```js
const { data, error } = await supabase
  .from('countries')
  .select('name, cities(name)')
  .eq('name', 'United States')
  .limit(1, { foreignTable: 'cities' })
```
