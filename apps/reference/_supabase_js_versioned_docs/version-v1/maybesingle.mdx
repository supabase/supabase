Retrieves at most one row from the result. Result must be at most one row
(e.g. using `eq` on a UNIQUE column), otherwise this will result in an
error.

```js
const { data, error } = await supabase
  .from('cities')
  .select('name, country_id')
  .eq('name', 'Singapore')
  .maybeSingle()
```

## Examples

### With `select()`

```js
const { data, error } = await supabase
  .from('cities')
  .select('name, country_id')
  .eq('name', 'Singapore')
  .maybeSingle()
```
