Generates email links and OTPs to be sent via a custom email provider.

```js
const { data, error } = await supabase.auth.admin.generateLink(
  'email@example.com'
  'signup',
  {
    'password': 'secret'
  }
)
```

```js

  'email@example.com'
  'signup',
  {
    'password': 'secret'
  }
)
```

const { data, error } = await supabase.auth.admin.generateLink(

  'invite',
)
