## Notes

- Requires a `service_role` key.
- This function should only be called on a server. Never expose your `service_role` key in the browser.

## Examples

### Basic example.

```js
const { data: user, error } = await supabase.auth.api.inviteUserByEmail(
  'email@example.com'
)
```
