Finds all rows whose value on the stated `column` exactly matches the
specified `value`.

```js
const { data, error } = await supabase
  .from('cities')
  .select('name, country_id')
  .eq('name', 'The shire')
```

## Examples

### With `select()`

```js
const { data, error } = await supabase
  .from('cities')
  .select('name, country_id')
  .eq('name', 'The shire')
```

### With `update()`

```js
const { data, error } = await supabase
  .from('cities')
  .update({ name: 'Mordor' })
  .eq('name', 'San Francisco')
```

### With `delete()`

```js
const { data, error } = await supabase
  .from('cities')
  .delete()
  .eq('name', 'Mordor')
```

### With `rpc()`

```js
// Only valid if the Postgres function returns a table type.
const { data, error } = await supabase
  .rpc('echo_all_cities')
  .eq('name', 'San Francisco')
```
