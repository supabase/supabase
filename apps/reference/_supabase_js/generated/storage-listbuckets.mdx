Retrieves the details of all Storage buckets within an existing project.

```js
const { data, error } = await supabase.storage.listBuckets()
```

## Notes

- Policy permissions required:
  - `buckets` permissions: `select`
  - `objects` permissions: none

## Examples

### List buckets

```js
const { data, error } = await supabase.storage.listBuckets()
```
