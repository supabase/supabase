Return `data` as a single object instead of an array of objects.

Query result must be one row (e.g. using `.limit(1)`), otherwise this
returns an error.

```js
const { data, error } = await supabase
  .from('cities')
  .select('name, country_id')
  .limit(1)
  .single()
```

## Examples

### With `select()`

```js
const { data, error } = await supabase
  .from('cities')
  .select('name, country_id')
  .limit(1)
  .single()
```
