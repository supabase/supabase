Log in a user given a User supplied OTP received via mobile.

```js
const { data, error } = await supabase.auth.verifyOtp({
  phone,
  token,
  type: 'sms',
})
```

## Examples

### Verify Sms One-Time Password (OTP)

```js
const { data, error } = await supabase.auth.verifyOtp({
  phone,
  token,
  type: 'sms',
})
```

### Verify Signup One-Time Password (OTP)

```js
const { data, error } = await supabase.auth.verifyOtp({
  email,
  token,
  type: 'signup',
})
```
