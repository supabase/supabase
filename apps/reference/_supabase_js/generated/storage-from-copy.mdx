Copies an existing file.

```js
const { data, error } = await supabase.storage
  .from('avatars')
  .copy('public/avatar1.png', 'private/avatar2.png')
```

## Notes

- Policy permissions required:
  - `buckets` permissions: none
  - `objects` permissions: `update` and `select`

## Examples

### Copy file

```js
const { data, error } = await supabase.storage

```
