Finds all rows whose value on the stated `column` doesn't match the
specified `value`.

```js
const { data, error } = await supabase
  .from('cities')
  .select('name, country_id')
  .neq('name', 'The shire')
```

## Examples

### With `select()`



```js
const { data, error } = await supabase
  .from('cities')
  .select('name, country_id')
  .neq('name', 'The shire')
```

### With `update()`



```js
const { data, error } = await supabase
  .from('cities')
  .update({ name: 'Mordor' })
  .neq('name', 'San Francisco')
```

### With `delete()`



```js
const { data, error } = await supabase
  .from('cities')
  .delete()
  .neq('name', 'Mordor')
```

### With `rpc()`



```js
// Only valid if the Postgres function returns a table type.
const { data, error } = await supabase
  .rpc('echo_all_cities')
  .neq('name', 'Lagos')
```
