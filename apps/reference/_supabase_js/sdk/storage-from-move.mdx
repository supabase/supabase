Moves an existing file.

```js
const { data, error } = await supabase
  .storage
  .from('avatars')
  .move('public/avatar1.png', 'private/avatar2.png')
```

## Notes

- Policy permissions required:
  - `buckets` permissions: none 
  - `objects` permissions: `update` and `select`
