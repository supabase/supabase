Downloads a file.

  defaultActiveId="dart"

<TabsPanel id="dart" label="dart">

```dart
final res = await supabase
  .storage
  .from('avatars')
  .download('avatar1.png');
```


</TabsPanel>

</Tabs>





## Notes

- Policy permissions required:
  - `buckets` permissions: none 
  - `objects` permissions: `select`
