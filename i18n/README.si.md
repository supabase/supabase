**Architecture**

Supabase ‡∂∫‡∂±‡∑î [hosted platform](https://supabase.com/dashboard). ‡∂î‡∂∂‡∂ß ‡∂ö‡∑í‡∑É‡∑í‡∑Ä‡∂ö‡∑ä ‡∑É‡∑ä‡∂Æ‡∑è‡∂¥‡∂±‡∂∫ ‡∂±‡∑ú‡∂ö‡∂ª ‡∂Ω‡∑í‡∂∫‡∑è‡∂¥‡∂Ø‡∑í‡∂Ç‡∂†‡∑í ‡∑Ä‡∑ì Supabase ‡∂∑‡∑è‡∑Ä‡∑í‡∂≠‡∑è ‡∂ö‡∑í‡∂ª‡∑ì‡∂∏ ‡∂Ü‡∂ª‡∂∏‡∑ä‡∂∑ ‡∂ö‡∑Ö ‡∑Ñ‡∑ê‡∂ö‡∑í‡∂∫.
‡∂î‡∂∂‡∂ß [self-host](https://supabase.com/docs/guides/hosting/overview) ‡∑É‡∑Ñ [develop locally](https://supabase.com/docs/guides/local-development) ‡∂ö‡∑Ö ‡∑Ñ‡∑ê‡∂ö‡∑í‡∂∫.

![Architecture](apps/docs/public/img/supabase-architecture.svg)

- [PostgreSQL](https://www.postgresql.org/) ‡∂∫‡∂±‡∑î ‡∂Ö‡∑Ä‡∑î‡∂ª‡∑î‡∂Ø‡∑î 30 ‡∑Ä‡∂©‡∑è ‡∂ö‡∑è‡∂Ω‡∂∫‡∂ö‡∑ä ‡∂ö‡∑ä‚Äç‡∂ª‡∑í‡∂∫‡∑è‡∂≠‡∑ä‡∂∏‡∂ö ‡∑Ä‡∑ô‡∂∏‡∑í‡∂±‡∑ä ‡∂¥‡∑Ä‡∂≠‡∑í‡∂± object-relational database system ‡∂ë‡∂ö‡∂ö‡∑ä ‡∑Ä‡∂± ‡∂Ö‡∂≠‡∂ª ‡∂ë‡∂∫ ‡∑Ä‡∑í‡∑Å‡∑ä‡∑Ä‡∑è‡∑É‡∂±‡∑ì‡∂∫‡∂≠‡∑ä‡∑Ä‡∂∫‡∂ß,‡∂ö‡∑ä‚Äç‡∂ª‡∑í‡∂∫‡∑è‡∂ö‡∑è‡∂ª‡∑ì‡∂≠‡∑ä‡∑Ä‡∂∫‡∂ß ‡∑Ñ‡∑è feature robustness ‡∂∂‡∑Ä‡∂ß ‡∂ö‡∑ì‡∂ª‡∑ä‡∂≠‡∑í‡∂∫‡∂ö‡∑ä ‡∂Ö‡∂≠‡∑ä‡∂¥‡∂≠‡∑ä‡∂ö‡∂ª‡∂ú‡∑ô‡∂± ‡∑É‡∑í‡∂ß‡∑í‡∂∫‡∑í
- [Realtime](https://github.com/supabase/realtime) is an Elixir server that allows you to listen to PostgreSQL inserts, updates, and deletes using websockets. Realtime polls Postgres' built-in replication functionality for database changes, converts changes to JSON, then broadcasts the JSON over websockets to authorized clients.
- [PostgREST](http://postgrest.org/) ‡∂∫‡∂±‡∑î PostgreSQL database ‡∂ö‡∑ô‡∂Ω‡∑í‡∂±‡∑ä‡∂∏ RESTful API ‡∂ë‡∂ö‡∂ö‡∑ä ‡∂∂‡∑Ä‡∂ß ‡∂¥‡∂ª‡∑í‡∑Ä‡∂ª‡∑ä‡∂≠‡∂±‡∂∫ ‡∂ö‡∂ª‡∂± web server ‡∂ë‡∂ö‡∂ö‡∑í.
- [GoTrue](https://github.com/netlify/gotrue) JWT tokens ‡∂∑‡∑è‡∑Ä‡∑í‡∂≠ ‡∂ö‡∂ª‡∂±‡∑ä‡∂±‡∂±‡∑ä‡∑Ä ‡∑Ñ‡∑ê‡∑É‡∑í‡∂ª‡∑Ä‡∑ì‡∂∏‡∂ß JWT ‡∂Ü‡∑Å‡∑ä‚Äç‡∂ª‡∑í‡∂≠ APi ‡∂ë‡∂ö‡∂ö‡∑í.
- [Storage](https://github.com/supabase/storage-api) S3 ‡∑Ä‡∂Ω ‡∂ú‡∂∂‡∂©‡∑è ‡∂ö‡∂ª ‡∂á‡∂≠‡∑í files ‡∑Ñ‡∑É‡∑î‡∂ª‡∑Ä‡∂±‡∑ä‡∂± RESTful interface ‡∂ë‡∂ö‡∂ö‡∑ä ‡∂Ω‡∂∂‡∑è‡∂Ø‡∑ì‡∂∏, Postgres ‡∂∑‡∑è‡∑Ä‡∑í‡∂≠ ‡∂ö‡∂ª‡∂∏‡∑í‡∂±‡∑ä ‡∂Ö‡∑Ä‡∑É‡∂ª‡∂∫‡∂±‡∑ä ‡∑Ñ‡∑ê‡∑É‡∑í‡∂ª‡∑Ä‡∑ì‡∂∏‡∂ß
- [pg_graphql](http://github.com/supabase/pg_graphql/) a PostgreSQL extension that exposes a GraphQL API
- [postgres-meta](https://github.com/supabase/postgres-meta) Postgres ‡∑Ñ‡∑ê‡∑É‡∑í‡∂ª‡∑Ä‡∑ì‡∂∏‡∂ß RESTful API ‡∂ë‡∂ö‡∂ö‡∑ä ‡∂Ω‡∂∂‡∑è‡∂Ø‡∑ô‡∂± ‡∂Ö‡∂≠‡∂ª table ‡∑Ä‡∂Ω ‡∂Ø‡∂≠‡∑ä‡∂≠ ‡∂Ω‡∂∂‡∑è‡∂ú‡∑ê‡∂±‡∑ì‡∂∏‡∂ß,roles add ‡∂ö‡∑í‡∂ª‡∑ì‡∂∏‡∂ß ‡∑Ñ‡∑è queries run ‡∂ö‡∑í‡∂ª‡∑ì‡∂∏ ‡∂ö‡∂ª ‡∂Ø‡∑ô‡∂∫‡∑í
- [Kong](https://github.com/Kong/kong) is a cloud-native API gateway.
