## Hiring for Postgres or Supabase Engineers?

If you're building with Postgres, you can add your job listing on our [Jobs Board](https://github.com/supabase/supabase/discussions/categories/jobs-board).

Make sure to format the tile in this format: 

```
{Company} | {Location|Remote} | {Full Time|Part Time|Contract} | {Role Name}
```

Include all the details which a candidate needs to contact you.
