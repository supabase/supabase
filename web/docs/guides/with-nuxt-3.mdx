## Intro 

This example provides the steps to build a simple todo app (from scratch!) using Supabase and Nuxt 3. It includes:

- Supabase [Database](/docs/guides/database): a Postgres database for storing your user data.
- Supabase [Auth](/docs/guides/auth): users can sign in with github provider.
- Instant [APIs](/docs/guides/api): APIs will be automatically generated when you create your database tables.

By the end of this guide you'll have an app which allows users to login and use our supabase hosted todo list.

You can check a [live demo](https://n3-supabase.netlify.app) of this project.

## Do it yourself


Just follow the [readme on Github](https://github.com/nuxt-community/supabase-module/tree/main/demo#todo-list-example-using-supabase-and-nuxt-3) to see the app in action.

This demo is using the [@nuxtjs/supabase](https://github.com/nuxt-community/supabase-module) module for first class integration with Supabase.

Checkout the module [documentation](https://supabase.nuxtjs.org/).
